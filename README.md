# GalaChain Claude Plugin Marketplace

Official marketplace for GalaChain Claude Code plugins.

## Quick Install

```bash
claude plugins add @gala-chain/claude-plugin
```

## Features

- **247 MCP Tools** - Comprehensive GalaChain operations
- **29 Categories** - Trading, liquidity, bridging, and more
- **41 Teaching Topics** - AI-powered learning
- **8 Learning Paths** - Structured skill development
