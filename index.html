Installation Steps

Two commands to activate the GalaChain Claude Code Plugin

Step 1: Add Marketplace
terminal
$ claude add marketplace aschulergala/claude-plugin-marketplace
Step 2: Add Plugin
terminal
$ claude plugins add @gala-chain/claude-plugin
Note: Requires Claude Code CLI. After running these commands, the plugin will automatically configure MCP servers and load all tools on activation. Restart Claude Code to see the new tools.
